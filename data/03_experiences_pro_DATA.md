# Expériences Professionnelles DATA

## Alternant — Développeur Python & Data Analyst
Groupama Centre-Atlantique — Sept. 2025 à Aujourd'hui (Niort)
Département : Administration des Ventes

- Technologies : Python, CustomTkinter, PyInstaller, VBA, CSV/SQL
- Réalisations principales :
	- Conception d'une application desktop (GUI CustomTkinter) centralisant la saisie des flux d'activité (mails, appels, dossiers) pour plusieurs équipes, remplaçant des dizaines de fichiers Excel individuels.
	- Packaging en exécutable autonome (`.exe`) via PyInstaller pour un déploiement "zero-config" sur postes utilisateurs.
	- Migration d'un modèle "fichiers annuels par agent" vers une structure de données unifiée (CSV structuré / SQL) réduisant la maintenance annuelle.
	- Refactoring des macros VBA pour permettre l'ingestion automatique des données dans les outils de reporting historiques.
- Impact : consolidation fiable des données, suppression de la maintenance annuelle, amélioration de la qualité et de la traçabilité des flux.
- Compétences transverses : gestion du changement, formation des utilisateurs, analyse des besoins fonctionnels.

### Projet — « Gestion des Flux »
- Type : Outil métier / Data engineering
- Problème : Multiplication de fichiers Excel entraînant erreurs de consolidation et maintenance lourde.
- Solution : Application desktop + centralisation des flux dans un format standardisé (CSV/SQL) + packaging `.exe` pour faciliter le déploiement.
- Résultat : Processus de suivi fiabilisé et maintenance simplifiée.

## Stage — Chargé d'Études Statistiques
CCI La Rochelle (Observatoire économique Viséo 17) — Avril 2025 à Juin 2025 (La Rochelle)

- Technologies : Excel, Access (SQL)
- Objectif : Optimiser le taux de réponse et réduire les coûts de l'enquête annuelle départementale.
- Réalisations principales :
	- Conception et mise en œuvre d'un plan d'échantillonnage stratifié sur une base de 40 000 entreprises (critères : secteur, taille, localisation).
	- Ciblage algorithmique des envois papier pour minimiser les coûts d'impression tout en maximisant le taux de réponse.
	- Nettoyage, normalisation et consolidation de bases hétérogènes ; développement de requêtes SQL complexes sous Access et solutions Excel pour contourner les contraintes techniques.
- Impact : réduction significative des coûts d'impression et production d'indicateurs fiables pour les décideurs locaux.
- Compétences transverses : rigueur méthodologique, adaptabilité aux contraintes techniques, synthèse des résultats.
