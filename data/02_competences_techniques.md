# Compétences Techniques et Outils

## Programmation et Développement Applicatif

* **Python** : Langage principal.
* *Développement* : Applications Desktop (**CustomTkinter**, `ttkbootstrap`), programmation asynchrone (**asyncio**), interaction API.
* *Déploiement* : Packaging d'exécutables autonomes (.exe) avec **PyInstaller**.
* *Data* : Scripting d'automatisation et analyse (Pandas).


* **VBA (Visual Basic for App)** : Refactoring de code legacy, interopérabilité Excel/Python, automatisation de flux complexes.
* **SQL** : Requêtes complexes, jointures, agrégations (MySQL, PostgreSQL, Oracle).
* **Web** : HTML5, CSS3, JavaScript, PHP (Interfaces et collecte de données).
* **Scripting Système** : **PowerShell** (Automatisation de tâches Windows).
* **R & Scala** : Analyse statistique et manipulation de dataframes (Notions académiques).

## Data Science et Business Intelligence

* **Manipulation de données** : Pandas, NumPy (Nettoyage, transformation et calcul matriciel).
* **Statistiques Avancées** : Techniques d'échantillonnage (Stratification, Aléatoire), redressement, inférence statistique.
* **Business Intelligence (BI)** :
* *Outils* : **Qlik Sense**, PowerBI, Tableau Software.
* *Réalisation* : Conception de dashboards interactifs et KPIs.
